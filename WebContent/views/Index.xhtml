Prova 3 (JSF)

Este é um projeto para demostrar algumas das habilidades que você aprendeu até agora na disciplina de PROG 3.

A tarefa consiste no seguinte:

  1. Seguindo o modelo MVC de desenvolvimento você deve criar um cadastro de disciplina (Model esta pronto). A disciplina esta detalhada dentro da página de Disciplinas. Além do cadastro o sistema deve apresentar uma lista das disciplinas cadastradas e seus respectivos professores, e deve permitir a edição de uma disciplina.
